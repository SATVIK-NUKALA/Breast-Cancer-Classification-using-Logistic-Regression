# Breast Cancer Classification using Logistic Regression
This project builds a **Machine Learning model** to classify breast cancer tumors as **Malignant** or **Benign** using Logistic Regression.
It uses the **Breast Cancer Wisconsin dataset** and applies data preprocessing, feature scaling, and model evaluation.

---

## Features

* Data preprocessing using Pandas
* Feature scaling using StandardScaler
* Logistic Regression model training
* Accuracy evaluation on training and test data
* Real-time prediction for new input data

---

## Dataset

* Source: `sklearn.datasets.load_breast_cancer`
* Total samples: 569
* Features: 30 numerical features
* Target:

  * `0` → Malignant
  * `1` → Benign
